加、减、乘、除

描述

这些运算符（分别）返回两人运算对象的和，差，积，商。这些操作受运算对象的数据类型的影响。所以，例如，9 / 4结果是2，如果9和2是整型数。这也意味着运算会溢出，如果结果超出其在相应的数据类型下所能表示的数。（例如，给整型数值767加1结果是-768）。如果运算对象是不同的类型，会用那个较大的类型进行计算。

如果其中一个数字（运算符）是float类型或double类型，将采用浮点数进行计算。

例子

y = y + 3;
x = x - 7;
i = j * 6;
r = r / 5;

语法

result = value1 + value2;
result = value1 - value2;
result = value1 * value2;
result = value1 / value2;

参数

value1: 任何变量或常量

value2: 任何变量或常量

提示

  • 要知道整型常量默认为int型，因此一些常量计算可能会溢出（例如：60 * 1000将产生负的结果）。

  • 选择一个大小足够大的变量以容纳你的最大的计算结果。

  • 要知道你的变量在哪一点将会“翻转”且要知道在另一个方向上会发生什么，例如：（0 - 1）或（0 - 768）。

  • 对于数学需要分数，就使用浮点变量，但是要注意它们的缺点：占用空间大，计算速度慢。

  • 使用强制类型转换符例如：（int）myFloat以在运行中转换一个变量到另一个类型。
