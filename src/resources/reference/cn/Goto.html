goto

在程序中转移程序流到一个标记点

语法

label:

goto label; // sends program flow to the label

提示

在C程序中不建议使用goto，而且一些C编程书的作者主张永远不要使用goto语句，但是明智地使用它可以简化某些代码。许多程序员不赞成使用goto的原因是，无节制地使用goto语句很容易产生执行流混乱的很难被调试程序。

尽管如此，仍然有很多使用goto语句而大大简化编码的实例。其中之一就是从一个很深的循环嵌套中跳出去，或者是if逻辑块，在某些条件下。

例子

for(byte r = 0; r < 255; r++)
{
    for(byte g = 255; g > -1; g--)
	{
        for(byte b = 0; b < 255; b++)
		{
            if (analogRead(0) > 250)
			{ 
				goto bailout;
			}
            // more statements ... 
        }
    }
}
